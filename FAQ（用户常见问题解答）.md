## 次数更新与扣除规则

**Q：会员次数是如何更新与扣除的？**

1. **更新节奏**：会员权益次数按「订阅周期」下发，与自然月无关。
2. **不累积**：当期未用完的次数在周期结束后直接清零，重新发放满额次数。
3. **举例**：小明在 12 月 1 日 15:00 开通 Andante（¥49/月），则 12 月 1 日 15:00 至 1 月 1 日 15:00 为一个周期，期间可用 10 次深度研究 + 10 次 Agent；若选择连续包月，1 月 1 日 15:00 立刻获得新的 10+10 次，与上月剩余无关。
4. **扣减规则**：任务执行成功后扣除次数。若任务执行失败，可通过页面「👎」反馈申请补次数，核实后将退还。

## 套餐升级与降级

**Q：升级会员如何计算差额？**

可随时升档，差额按天折算。升级时，原套餐按当前订阅期剩余天数折算退费，新套餐从升级当天起计算新订阅期。

**Q：订阅错了套餐，想升级/降级怎么办？**

- **升级**：直接在会员界面点击目标套餐订阅，差额按天折算。
- **降级**：Kimi 暂不支持直接降级，请先取消自动续费，待当前订阅到期后再订阅所需套餐。

**Q：无法升级会员怎么办？**

可能是你已在其他渠道订阅了会员。请前往**初次支付的平台**进行升级管理。

以微信支付为例：【我的】→【钱包】→【账单】→ 搜索「月之暗面」→ 找到订单 →【账单服务】→【管理扣费服务】。

如需进一步帮助，请邮件联系 [membership@moonshot.cn](mailto:membership@moonshot.cn)。

**Q：如何取消连续自动订阅？**

成功退订后，当期权益不受影响，到期后不再续订。各渠道操作路径如下：

# Kimi 会话产品

---

## 对话异常

**Q：对话发送不出去/发送显示红圈怎么办？**

可能由以下原因导致：

- **触发频次限制**：Kimi 在一定时间段内存在对话上限，短时间内频繁发送可能导致无法继续。建议间隔 2-3 小时后再试，并注意控制对话频次。
- **网络状况不佳**：手机信号弱或网络不稳定时，消息可能无法发出。建议切换网络环境后重试。

**Q：会话超 20 万字了怎么办？**

单轮上下文上限约为 128K token（约 20 万汉字），达到上限后模型无法继续读入新内容。建议：

1. **新建对话**：将原对话的核心结论或提纲复制到新对话中，即可延续。
2. **生成交接文档**：让 Kimi 先「总结并生成交接文档」，再粘贴到新对话作为初始上下文，可节省字数并保留信息。

> **提示**：
> - 删减同一对话下的历史消息并不会释放上下文窗口，请直接开新会话。
> - Thinking 模型由于推理会占据更多 token，上下文上限更小。处理长文档时，优先选用 K2 而非 K2-Thinking。

**Q：Kimi 出现「换个话题聊聊吧」提示？**

该提示表示当前内容触发了安全策略，当前话题无法继续。直接输入新主题即可重新开启对话，无需等待。若确认内容无违规，可点击「👎」反馈，帮助我们优化安全策略。

**Q：Kimi 回复「Kimi 有点累了，可以晚点再问我一遍」？**

当前在线人数过多，算力资源紧张，属于高峰期限流。可稍等 1-2 分钟后重试。如需减少等待，可考虑开通会员订阅，享受专属算力通道优先服务。

# Agent

---

## 功能与限制

**Q：Agent 能做什么？**

Agent 通过调用多种工具满足以下需求：

- **文本生成与编辑**：生成文章、报告、文档等，支持多种写作风格和格式。
- **网页开发**：编写 HTML，支持 APP 界面设计、静态网页部署、数据可视化等。
- **PPT 生成**：拥有独立的网页版 PPT 工具，可生成完整演示文稿。
- **辅助编程**：支持多种编程语言的代码编写与调试。

**Q：为什么 Agent 做出来的网页不能用？**

一个完整的网页通常包含前端界面、后端服务和数据存储三部分。Agent 目前专注于**静态网页**的生成与部署，适合展示类页面、数据可视化等场景。

若需求涉及用户登录、数据存储、动态交互等后端功能，Agent 暂无法直接实现。建议先用 Agent 完成前端部分，再搭配其他工具补充后端功能。

**Q：Agent 的任务交付物是什么？**

- 完整的任务执行文件
- 部署好的静态网页
- 可下载的 Zip 包（位于 `/mnt/okcomputer/output/` 目录）

**Q：Agent 的上下文长度是多少？**

Agent 使用的 context length 为 **128K tokens**（约 20 万字）。

## 额度与计费

**Q：Agent 额度用完，如何续费？**

可升级会员订阅套餐，获得新套餐对应的次数额度。

**Q：Agent 没有完成任务就扣除了次数？**

Agent 对话分为「普通对话」与「执行任务」两类，选择「Agent」任务后每次输入均会消耗一次付费额度，与 token 消耗量无关。

- **计费逻辑**：按输入次数计算，与 token 消耗量无关。
- **任务失败退还**：若任务执行失败（未返回有效结果），可点击「👎」反馈，工作人员核实后退还相应次数。

# 深度研究

---

## 任务执行

**Q：深度研究「卡住了」/「一直没有出结果」怎么办？**

深度研究会进行大量网页检索、资料验证与推理，界面未及时刷新不代表任务中断。可尝试：

1. 刷新浏览器页面；
2. 暂时离开当前会话（不影响任务执行）；
3. 稍后返回查看，任务仍在后台运行。

若刷新后仍无结果，请检查网络连接是否稳定后再等待片刻。

**Q：我发现深度研究搜索或思考方向出现了错误？**

若发现搜索方向或推理结论明显偏离预期，可立即点击页面底部「👎」按钮，在弹窗中简要说明「问题类型 + 具体偏差」（如：搜索偏离主题 / 结论与事实不符）。平台将在 1-3 个工作日内完成核查，确认属实时自动退还本次次数。

**Q：深度研究的上下文长度是多少？**

深度研究使用的 context length 为 **128K tokens**（约 20 万字）。

---

## 额度与保存

**Q：深度研究没有完成任务就扣除了次数？**

深度研究同样区分「普通对话」与「执行任务」，选择「深度研究」任务后即按任务类型计费。若任务执行失败，可点击「👎」反馈申请退还次数。

**Q：我主动停止了任务，却被扣除了次数？**

深度研究一旦启动，系统即调用算力进行检索、验证与推理，资源已被锁定，因此主动点击「停止」或关闭页面均视为已消耗次数。

若页面长时间无更新，建议先刷新浏览器或稍后再查看，避免因误判「卡住」而中断任务。如确认为系统异常，可通过「👎」反馈，人工核实后退还次数。

**Q：深度研究报告无法下载和保存？**

深度研究生成两种报告形式，均可保存：

1. **文字研究报告**：点击右上角「下载」或「复制」按钮，支持本地保存或粘贴使用。
2. **可视化报告**：点击「预览」→「分享」→「复制链接」→ 在浏览器中打开 → 右键「另存为」，可保存为 HTML、PDF 等格式。

# Kimi API

---

## 错误排查

**Q：调用 API 时报错 400 / 401 / 403 / 404 / 429 / 500 怎么办？**

请根据错误码对照以下说明排查：

| 错误码 | 含义 | 建议处理方式 |
|-------|------|------------|
| 400 | 请求参数错误 | 检查请求体格式与参数是否符合文档要求 |
| 401 | 认证失败 | 重新创建 API Key，检查传递方式是否正确 |
| 403 | 权限不足 / 余额不足 | 确认使用的是正确账号的 Key；检查余额是否足以覆盖本次请求 |
| 404 | 资源不存在 | 检查模型名称、接口路径是否正确 |
| 429 | 速率限制 | 请求数达到账户等级上限，或高峰期资源受限，稍后重试 |
| 500 | 服务端错误 | 稍后重试；若持续出现，请联系客服 |

**Q：API 认证失败（Invalid Authentication / 401 错误）怎么办？**

建议重新创建一个 API Key，并参照官网示例代码确认传递方式是否正确。

**Q：账户还有余额，API 返回 403 是什么原因？**

请确认使用的是正确账号的 API Key；另外，也可能是本次请求所需 Token 消耗超过了当前余额。

**Q：API 返回 429 速率限制错误如何解决？**

可能是请求数达到了当前账户等级的上限，也可能是高峰期资源受限。可稍后重试；如需更高速率，请联系人工客服申请提升。

**Q：API 返回结果不完整被截断（finish_reason=length）怎么办？**

原因通常是未设置 `max_tokens`，导致输出超过默认长度限制。若已设置仍截断，可能是超出了所用模型的最大输出上限，建议切换至支持更长输出的模型。

## 计费与余额

**Q：Kimi API 是怎么计费的？**

按输入和输出消耗的 Token 计算费用，不同模型价格不同。详情请查阅：[定价说明](https://platform.moonshot.cn/docs/pricing/chat)

- 联网搜索：触发 `$web_search` 工具调用计费一次，每次 ¥0.03。详见：[工具定价](https://platform.moonshot.cn/docs/pricing/tools)
- Context Caching 定价同样在上述页面中说明。

> 表格中价格代表每消耗 1M tokens（= 1,000,000 tokens）的费用。

**Q：我的余额怎么没了 / 为什么消耗这么快？**

可在 [费用明细页面](https://platform.moonshot.cn/console/fee-detail) 查看账户消耗的 Token 数、请求时间和使用模型（注：当日账单在次日 7 点更新，无法查看请求具体内容）。

- 计算 Token 数量：[Token 估算接口](https://platform.moonshot.cn/docs/api/estimate)
- 查询账户余额：[余额查询接口](https://platform.moonshot.cn/docs/api/balance)

**Q：文件上传不是免费的吗？为什么仍产生了费用？**

文件上传和内容抽取功能均免费。但将抽取内容作为输入发送给模型进行 Chat，会消耗 Token 并产生费用。详见：[文件问答指南](https://platform.moonshot.cn/docs/guide/use-kimi-api-for-file-based-qa)

**Q：Allegretto 订阅用户，为什么 API 并发数仍受限 / 仍显示 Tier 0？**

Kimi 会员订阅提供的是 **Kimi For Coding** 的调用次数，与开放平台 API 调用不互通。如需配置 Kimi For Coding，请参阅：[Kimi For Coding 文档](https://www.kimi.com/coding/docs/)

**Q：我要申请 API 账户退款。**

请联系 API 客服处理。

## 模型与功能

**Q：Kimi 最快 / 上下文最长的模型是哪个？各模型有什么区别？**

不同模型的性能介绍请查阅：[模型介绍文档](https://platform.moonshot.cn/docs/introduction)

- Vision 模型均支持图片理解（图片固定消耗 1024 Tokens）。
- 目前暂不支持视频多模态输入。
- 暂无 PPT 助手 / 深度研究的 API 接口。

**Q：如何提速 / 提升 TPM、RPM、TPD 或并发数上限？**

速率限制基于账户累计充值金额分级管理。如有更高需求，请联系人工客服申请。

**Q：调用 API 返回结果不如 Kimi 网页版，为什么？**

可能原因：

1. 使用的模型与网页版不一致。建议调用 `kimi-latest`（无后缀），该模型始终对应 Kimi 智能助手所用的最新版本。
2. 网页版对 system prompt 做了大量优化，建议在 API 调用时同步优化你的 system prompt。

**Q：可以使用文件接口上传图片吗？**

可以，但标准模型的文件接口仅支持对图片进行 OCR 识别，不支持图片内容理解。如需理解图片语义，请切换至视觉模型（如 `moonshot-v1-8k-vision-preview` / `moonshot-v1-32k-vision-preview` / `moonshot-v1-128k-vision-preview`）。

**Q：调用联网搜索 API，能返回搜索结果来源吗？**

目前出于安全合规原因，搜索结果来源暂不对外显示。

**Q：如何让模型稳定输出 JSON 格式？**

可启用 JSON Mode，将 `response_format` 参数设置为 `{"type": "json_object"}`，模型将输出合法且可被正确解析的 JSON。详见：[JSON Mode 使用说明](https://platform.moonshot.cn/docs/guide/use-json-mode-feature-of-kimi-api)

**Q：如何删除通过文件接口上传的文件？**

参考文档中的操作方法：[文件接口文档](https://platform.moonshot.cn/docs/api/files)

**Q：Kimi 是否支持微调 / 本地化部署 / TTS / ASR？**

| 功能 | 是否支持 |
|-----|---------|
| 微调 | 暂不支持 |
| 本地化部署 | 暂不支持，有需求请[联系销售](https://platform.moonshot.cn/contact-sales) |
| TTS / ASR 接口 | 暂不支持 |

**Q：国外能调用 Kimi API 吗？**

可以，请使用专属国际平台接口：[国际平台文档](https://platform.moonshot.ai/docs/overview)

## 账号与认证

**Q：个人认证与企业认证有什么区别？**

企业认证支持邀请成员加入企业，并体验企业相关功能。其他区别详见：[认证对比说明](https://platform.moonshot.cn/docs/guide/org-best-practice)

- 个人认证可转为企业认证。
- 企业认证**不支持**转回个人认证。
- 企业认证用户同样支持微信 / 支付宝方式充值，无需对公转账。
- 企业认证用户**可以**申请增值税普通发票或专用发票，[自助开票入口](https://platform.moonshot.cn/console/invoice)。

**Q：我的账号被封了，如何申诉？**

请发邮件至 [api-service@moonshot.cn](mailto:api-service@moonshot.cn)。

**Q：原注册手机号已注销，如何换绑？**

请联系 API 客服，并提供以下材料：

- 曾绑定的手机号（即使已注销）
- 手机号注销证明（如运营商出具的销户凭证）
- 可证明你为开放平台持有者的信息（如充值记录、订单截图等）

客服团队核实后将协助完成换绑。
