# Kimi Websites 网站

## 1. Kimi Websites 网站是什么
Kimi Websites 网站是 Kimi Agent 产品功能，由最新的 Kimi K2.5 多模态视觉代理 AI 模型支持。核心产品能力是可视化编程（Vibe Coding），即用自然语言进行网站生成，并进行在线部署。利用模型对视觉信息（如 UI 设计图、截图、交互视频）与自然语言指令的深度理解，将用户输入生成执行为完整、可运行的前端代码，生成代码不仅覆盖静态 UI，还能具备动态交互行为。包括：

- HTML 页面结构
- CSS 样式布局
- JavaScript 交互逻辑（如滚动触发效果、动效动画等）

### 产品功能说明

| 功能           | 说明                                                       |
|----------------|------------------------------------------------------------|
| 对话式建站     | 用自然语言描述需求，AI 自动生成网站                         |
| 图片/视频理解 | 上传设计图/截图/录屏，AI 自动解析并生成 HTML 页面            |
| 自动部署       | 支持一键自动部署，快速上线                                 |
| 多轮修改       | 支持网页版本多轮修改，并查看不同版本效果                   |
| 一键分享       | 生成可访问的链接，方便分享展示                             |
| 模板选择       | 提供多种预设模板，快速开始建站                             |
| 代码导出       | 支持打包下载 HTML/CSS/JS 文件，自主部署                    |

## 2. 如何使用 Kimi Websites 网站功能

### 方法一：访问 Kimi Websites 专属页面
<img width="2582" height="1428" alt="image" src="https://github.com/user-attachments/assets/911dc731-78bd-4212-add6-706d1b8647f2" />

1. 打开浏览器：使用 Chrome、Safari、Edge、Firefox 等主流浏览器，
2. 打开官方访问地址：Kimi Websites 专属页面：[https://www.kimi.com/zh/websites](https://www.kimi.com/zh/websites)（支持中文/英文两种界面）
3. 登录账号：使用 Kimi 账号登录（支持手机号、邮箱注册）
4. 输入建站任务，Kimi Websites 提供三种主要建站方式：
   1. **文字输入**：在对话框中输入任务，用自然语言描述想要的网站效果；
   2. **图片/网站复刻**：通过上传设计稿/喜欢网站的截图触发对应功能；
   3. **模板建站**：快速搭建标准网站，选择预设模板，修改内容即可；
5. 开始建站：进入 Websites 功能界面，选择模板或直接对话创建。

### 方法二：移动 App 选择专属网站工具

iOS/Android/鸿蒙 均有官方 App（应用名“Kimi”），在输入框上方，点击工具栏（Taskbar）→ 切换至网站（Websites）Agent 模式，内含多模态生成工具与 Agent 功能，可进行视觉输入并生成前端代码。

### 方法三：在通用 Agent 模式下，输入建站任务需求
<img width="1280" height="877" alt="image" src="https://github.com/user-attachments/assets/a13d1739-52fa-4450-9b18-b84e2a0279e8" />

在官网或移动设备，切换到 K2.5 Agent 模式，并输入建立网站相关的任务指令，Kimi 会调用视觉编程能力和代码执行工具。

## 3. Kimi Websites 网站如何运行?

### 第一步：对话创建网站

在输入框中用自然语言描述需求，Kimi 即可理解并执行。支持三种创建方式：

| 方式               | 说明                                      |
|--------------------|-------------------------------------------|
| 文本描述           | 直接输入网站需求，AI 自动生成             |
| 设计图解析         | 上传设计稿，AI 识别并还原为网页           |
| 快速搭建标准网站   | 选择预设模板，修改内容即可               |

### 实际案例演示：
```
输入建站任务
为西高地小狗打造一个网站，主色调黑白。包含三个栏目：
1. 影棚级写真日历，AI 生成图片展示；
2. 「卷毛实验室」养护百科；
3. 周边商城，销售西高地头像卫衣、限量银箔海报。结账页需嵌入公益模块，每售出 1 件捐 10 元给流浪小狗救助。
```

### 第二步：建站流程概览
```
需求描述 → Kimi 进行思考 → 调用 webapp-building SKILL →
了解技术栈和构建流程 →创建任务列表并初始化项目 → 编写待办清单 → 运行终端 → 生成图片 → 预览调整 →
更新任务进度，开发网站核心代码 → 更新 CSS 和配置文件，然后创建组件 → 多轮修改 → 部署上线 → 分享传播
```
<img width="1906" height="1088" alt="image" src="https://github.com/user-attachments/assets/81c83cf2-5395-4b83-929c-e975b8fb46b4" />

Kimi Website 会激活 webapp-building SKILL，这是一个专门用于网站构建的多轮代码生成代理，用于自主执行流程：

1. 需求解析：自动分析网站功能模块、视觉风格、交互逻辑
2. 任务规划：自主拆解为设计、开发、素材准备等子任务
3. 技术方案：自动选择适配的技术架构（如 React + Tailwind CSS）
4. 素材生成：按需生成图片、图标等视觉素材
5. 代码构建：编写页面结构、样式与交互代码
6. 多轮优化：根据预览效果自主调整，直至完成

### 第三步：预览与调整

- **实时预览**：生成 HTML 演示地址，可立即在浏览器中查看效果
- **对话修改**：继续对话要求调整，如“把导航栏改成深色”

#### 真实案例：
```
颜色不太好看，我希望去掉黑白的要求，变成一个正常的网站。  
Kimi Websites 会重新编排待办清单，并继续构建网站。
```

### 第四步：可视化在线编辑
<img width="902" height="234" alt="image" src="https://github.com/user-attachments/assets/9c0ebfb2-f761-409d-a2e8-72b4c4719f5e" />

- **多轮优化**：通过持续对话精细调整网站细节
- **修改方式**：
  - **标注模式**：在网页上，点击需要修改的地方，用自然语言添加批注
  - **选择模式**：可以选择网页上的任意元素，并添加批注
  - 通过矩形、箭头、画笔、切换颜色等多种选择框，修改工具，添加修改建议
- 在多处完成批注后，将批注添加到对话，一次性给 Kimi 发出修改建议

### 第五步：最新版代码部署与发布
| 部署方式   | 说明                                                 |
|------------|------------------------------------------------------|
| 自动部署   | Kimi 提供一键部署服务，生成可访问的在线链接          |
| 下载部署   | 打包下载代码文件，自行部署到服务器                  |
| 绑定域名   | 将生成的网站绑定到域名                              |

生成完毕后进入预览页面，  
可在预览界面，点击不同部署版本，下载不同版本文件。
