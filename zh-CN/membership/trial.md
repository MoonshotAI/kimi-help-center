# Andante 七天试用权益介绍

Kimi 为新用户提供 Andante 套餐的 **7 天免费试用**，让你在正式订阅前充分体验会员权益。

## 试用包含的权益

试用期间，你将享有 Andante 套餐的全部权益：

- 更多 Agent 额度
- Agent 4 倍速优先使用
- K2.5 模型提速并提升额度
- Kimi Code 可调用

## 如何开通试用

1. 打开 Kimi，进入**设置 → 订阅**页面。
2. 在 Andante 套餐下方点击「免费试用 7 天」。
3. 按提示完成支付方式绑定（试用期内不会扣费）。
4. 开通成功后即刻生效，可立即使用全部会员功能。

## 试用期注意事项

- **随时取消**：试用期内可随时在订阅管理页面取消，取消后不会产生任何费用。
