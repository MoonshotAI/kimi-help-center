# 取消自动续费

如果你不再需要 Kimi 会员服务，可以随时取消自动续费。取消后，当前计费周期内的会员权益仍然有效，到期后将不再续费。

## 如何取消连续自动订阅？

根据你的订阅渠道，选择对应的取消方式：

### 通过 Kimi 页面取消

1. 登录 Kimi（网页版或手机 App）。
2. 进入「设置 → 订阅管理」。
3. 点击「取消自动续费」或「管理订阅」。
4. 确认取消即可。

### 通过微信取消

1. 打开微信，进入「我 → 服务 → 钱包」。
2. 点击右上角「账单」→搜索「月之暗面」→「管理扣费服务」（或「自动续费」）。
3. 点击找到扣费账单。
4. 点击「关闭扣费服务」并确认。

### 通过支付宝取消

1. 打开支付宝，进入「我的 → 设置」。
2. 点击「支付设置 → 免密支付/自动扣款」。
3. 找到「Kimi 会员」，点击进入。
4. 点击「关闭服务」并确认。

### 通过 Apple ID 取消（iOS 用户）

1. 打开 iPhone「设置」。
2. 点击顶部你的 Apple ID 头像。
3. 进入「订阅」。
4. 找到「Kimi」订阅，点击「取消订阅」。

### 通过 Mac 取消

1. 打开 Mac 上的 App Store。
2. 点击左下角你的头像 → 「账户设置」。
3. 在「订阅」栏找到「Kimi」，点击「管理」。
4. 选择「取消订阅」并确认。

<Callout type="tip">
取消自动续费后，当前订阅周期内的权益将保留至到期日。如需重新订阅，随时可在 Kimi 订阅页面操作。
</Callout>
