# 联系与反馈

## 反馈渠道

如果你在使用 Kimi Code 的过程中遇到问题或有建议，可以通过以下方式反馈。

### 方式一：在 CLI 中直接反馈

在 Kimi Code CLI 的交互模式中，输入 `/feedback` 命令即可提交反馈。系统会自动收集当前环境信息，帮助你更准确地描述问题。

### 方式二：GitHub Issues

访问 [Kimi CLI GitHub Issues](https://github.com/MoonshotAI/kimi-cli/issues) 页面，搜索是否已有类似问题。如果没有，可以创建新的 Issue 详细描述你遇到的情况。

### 方式三：邮件反馈

你也可以通过邮件向 Kimi Code团队[code@moonshot.ai](mailto:code@moonshot.ai)反馈问题。请在邮件中尽量包含以下信息：

- **问题描述**：清晰描述你遇到的问题或期望的功能
- **复现步骤**：如何一步步触发这个问题
- **环境信息**：操作系统、Kimi Code CLI 版本（可通过 `kimi --version` 查看）
- **错误日志**：如果有报错信息，请完整粘贴

## 常见问题自助排查

在提交反馈之前，建议先检查以下几点：

- 是否已更新到最新版本的 Kimi Code CLI（运行 `kimi --version` 查看）
- 网络连接是否正常，能否访问 `api.kimi.com`
- API Key 是否有效且未过期
- 工作目录是否正确，Kimi 是否有权限访问相关文件
