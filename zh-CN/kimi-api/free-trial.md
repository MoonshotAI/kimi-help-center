# 免费测试权益

Kimi API 为新注册用户提供免费测试额度，让你在正式使用前充分体验平台能力。

## 赠送规则

- **适用对象**：国内手机号注册的新用户。
- **赠送金额**：注册即赠送 **15 元代金券**。
- **有效期**：代金券自发放之日起 **3 个月**内有效，过期自动失效。

## 代金券使用说明

- 代金券自动应用于 API 调用消耗，无需手动激活。
- 代金券余额优先于充值余额扣减。
- 可在控制台「费用明细」页面查看代金券剩余额度。

## 15 元能做什么？

15 元代金券足够进行大量的开发测试，例如：

- 数百次标准对话请求
- 数十次长文档解析
- 充分的接口联调和功能验证

具体可用次数取决于所选模型和每次请求的 token 消耗量。

## 代金券用完后

代金券用完后，API 调用将从充值余额中扣费。如余额为零，API 请求将返回 403 错误。你可以在控制台随时充值，充值后立即可用。
