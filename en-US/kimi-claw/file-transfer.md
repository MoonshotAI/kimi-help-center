Kimi Claw supports sending and receiving files.

## Send files to Kimi Claw

You can send images, documents, and other files to Kimi Claw through Kimi. It will use its built-in tools to view and process them.

## Receive files from Kimi Claw

The latest version of Kimi supports Kimi Claw sending files back to you. If file delivery isn't working, make sure you're running the latest version of the Kimi plugin.

## Manage files in the workspace

You can also browse and download files generated by Kimi Claw through the **Workspace** on the web. The Workspace stores all files produced during Kimi Claw's operation, available for you to access anytime.
