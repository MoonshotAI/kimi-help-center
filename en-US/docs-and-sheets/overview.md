## Kimi Docs
Kimi Docs has strong text adaptation capabilities and systematic understanding of Office workflows.
### Key capabilities
**Word Documents:**
- Generate long-form professional documents: white papers, research reports, contracts, proposals
- Insert comments and revision suggestions into existing Word documents
- Handle heading hierarchies, paragraph spacing, citations, and list formatting
- Condense a 30,000-word thesis into a precise summary
**PDF Documents:**
- Generate richly illustrated professional PDF reports with covers, table of contents, body text, and figure captions
- Re-edit and reformat existing PDFs without professional typesetting software
- Extract specific information from hundred-page PDFs
- Compare subtle differences across multiple contract versions
### How to use Kimi Docs？
**Entry Points:**
- General: Click Docs, Sheets, or Slides shortcuts in the sidebar or chat box
- Dedicated: Click "Docs" button at the bottom of the chat box
### Method 1: visit the Kimi Docs page

1. **Open browser**: Chrome, Safari, Edge, Firefox, or any major browser
2. **Go to**: [kimi.com/docs](https://www.kimi.com/docs)
3. **Log in**: Sign in with your Kimi account (Google or email)
4. **Enter a task**:
   - **Text input**: Type a task in natural language
   - **Upload a document** for editing
   - **Browse examples**: Click "Make one like this," customize, then send
5. **Start creating**: Choose a template or create via conversation

Describe your needs directly, e.g., "Create an XX report for me." Or upload an existing Word/PDF/Excel file and specify operations. Download the original-format file (.docx / .xlsx / .pdf) to continue editing locally.

### Method 2: mobile app
The Kimi app is available on iOS and Android. Tap the Taskbar → switch to Docs Agent mode.
#### Method 3: general Agent mode
Switch to K2.5 Agent mode and enter a document-related task. Kimi will invoke reading and writing tools.
## Kimi Sheets
Kimi Sheets understands cell relationships, cross-sheet linkage logic, and produces .xlsx files directly in the browser.
### Key capabilities
| Capability | Description |
|------------|-------------|
| **Bulk data entry** | Extract data from documents — e.g., pull exhibitor info from PDF into a 1,000-row Excel file |
| **Multi-source consolidation** | Merge 50 different department Excel reports into a single company summary |
| **Natural language input** | Autonomous web search with intelligent spreadsheet generation |

### How to use Kimi Sheets？
Click the "Agent" button at the bottom of the chat box, or select the "Sheets" shortcut in the sidebar.

<Frames
  src="./images/overview/screenshot-4.png"
  alt="screenshot 4"
/>

You can say: *"Search the web and compile the latest publications on XX field into an Excel document,"* or upload an existing Excel/CSV file and specify operations. Preview online, then download the .xlsx file.
