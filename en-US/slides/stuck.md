# What if Slides generation gets stuck?

<Callout type="info">
If the generation process stalls and the file never completes, it is usually caused by a Slides generation failure.
</Callout>

## Solution

<Callout type="warning">
Start a new conversation and re-run the Slides tool with the same instructions.
</Callout>

## Steps

1. Close the stuck conversation
2. Open [Kimi Slides](https://www.kimi.com/slides) and start a new conversation
3. Re-enter your slides generation request
4. Wait for the system to complete generation
