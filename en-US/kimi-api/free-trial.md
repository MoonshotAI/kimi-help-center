Kimi API offers free trial credits to newly registered users, allowing you to fully explore the platform's capabilities before committing.

## Trial credits

- **Eligibility**: New users who register with an email address.
- **Credit Amount**: A **$2 credit voucher** is granted upon registration.
- **Validity**: The voucher is valid for **3 months** from the date of issuance and expires automatically thereafter.

## Use your voucher

- The voucher is automatically applied to API usage — no manual activation required.
- Voucher balance is consumed before any topped-up balance.
- You can check your remaining voucher balance on the "Billing Details" page in the console.

## What can $2 get you?

The $2 trial credit is sufficient for extensive development testing, for example:

- Hundreds of standard chat requests
- Dozens of long-document parsing calls
- Thorough API integration testing and feature validation

The exact number of calls depends on the model selected and the token consumption per request.

## After the voucher is used up

Once the voucher is depleted, API calls will be charged against your topped-up balance. If your balance is zero, API requests will return a 403 error. You can top up at any time in the console — funds are available immediately.
