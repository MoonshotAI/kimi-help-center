The Kimi API platform provides developers with powerful large language model capabilities, helping you quickly integrate AI into your products and services.

## Platform overview

Kimi API is powered by Moonshot's proprietary large language models, offering text generation, multi-turn conversations, file parsing, web search, and other core capabilities. The platform is compatible with the OpenAI API format, enabling developers to migrate and integrate with minimal effort.

**Platform URL**: [platform.moonshot.ai](https://platform.moonshot.ai)

## Target audience

- **Independent Developers**: Quickly build AI-powered personal projects and tools.
- **Enterprise Engineering Teams**: Embed AI capabilities into enterprise products to improve user experience and operational efficiency.
- **Startups**: Access top-tier AI capabilities at low cost to accelerate product iteration.

## Key use cases

- **Intelligent Customer Service**: Context-aware automated Q&A to improve support efficiency and customer satisfaction.
- **Content Generation**: Article writing, marketing copy, code generation, and other creative workflows.
- **Data Analysis**: Document interpretation, report analysis, knowledge extraction, and summarization.
- **Knowledge Q&A**: Intelligent retrieval and question-answering systems powered by private document repositories.

## Quick start

1. Visit [platform.moonshot.ai](https://platform.moonshot.ai) to register a developer account.
2. Create an API Key in the console.
3. Refer to the API documentation to make your first call.
4. Choose the appropriate model and parameters for your use case.
