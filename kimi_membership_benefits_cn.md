# 会员权益说明

## 概述

Kimi 会员是一项灵活的订阅计划，Kimi提供包月与包年两种订阅方式，用户可以根据使用需求选择最合适的订阅计划。

Adagio 为免费版用户权益，会员权益（按周期计费，自动续订）分为 4 档：
Andante，入门版用户权益；
Moderato，基础版用户权益；
Allegretto，高级版用户权益；
Allegro，进阶版用户权益。

## Adagio（免费）权益介绍

权益内容
免费获得 3 次 Agent 的使用额度；
免费获得 1 次深度研究的使用额度；
免费获得 3 次 PPT 的使用额度；

可正常通过网页、APP 等端口访问并使用 Kimi 的部分模型（以实际提供为准）；
支持联网搜索能力。

**关于 Agent 使用额度**： Kimi 中的 Agent、网站、文档、表格等功能（不包括深度研究、PPT功能），统一归属于 Agent 功能。用户的 Agent 使用额度可在这些功能之间共享使用，即用户使用上述任一功能均会消耗 Agent 使用额度。不同功能因任务复杂度不同，可能消耗的额度有所差异，具体以实际使用时的页面提示为准。

**关于 Kimi Claw**：Kimi Claw 与 Kimi Code 共享可用额度，按照会员档位下发对应额度。如会员过期，Kimi Claw 云资源将为用户保留七天，之后会永久关停并释放资源，届时用户的相关数据和资源都将清空，请你务必注意。不支持同时运行两个及以上的 Kimi Claw 任务。
